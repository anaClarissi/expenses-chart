## 📖 Sobre o Projeto

Este projeto apresenta um gráfico de despesas dos últimos sete dias, exibindo visualmente os gastos diários através de barras proporcionais aos valores recebidos de um arquivo JSON.

O desafio teve como objetivo praticar a manipulação de dados com JavaScript, a criação dinâmica de elementos HTML e a construção de interfaces responsivas e acessíveis.

---

## 🎯 Objetivo

Durante o desenvolvimento foram praticados conceitos como:

* Consumo de dados externos em formato JSON.
* Manipulação do DOM.
* Criação dinâmica de componentes.
* Cálculo proporcional para geração de gráficos.
* Responsividade.
* Acessibilidade básica.

---

## ✨ Funcionalidades

* ✅ Leitura de dados a partir de arquivo JSON.
* ✅ Geração dinâmica das barras do gráfico.
* ✅ Destaque automático para o dia atual.
* ✅ Exibição do valor gasto ao passar o mouse ou navegar pelo teclado.
* ✅ Layout responsivo.
* ✅ Navegação acessível por teclado.
* ✅ Feedback visual para interação do usuário.

---

## ♿ Acessibilidade

Algumas práticas de acessibilidade implementadas:

* Uso de atributos ARIA.
* Navegação por teclado através de `tabindex`.
* Informações descritivas com `aria-label`.
* Estrutura semântica para leitura por tecnologias assistivas.
* Destaque visual para elementos em foco.
